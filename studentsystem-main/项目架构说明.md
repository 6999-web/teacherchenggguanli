## 🔄 数据流向

### 用户登录流程

```
1. 用户输入学号密码
   ↓
2. 前端发送 POST /api/auth/login
   ↓
3. 后端验证凭据
   ↓
4. 生成 JWT Token
   ↓
5. 返回 Token 和用户信息
   ↓
6. 前端存储 Token
   ↓
7. 后续请求携带 Token
```

### 数据查询流程

```
前端组件
   ↓ axios.get('/api/achievements')
后端路由 (routers/student.py)
   ↓ 依赖注入：JWT 验证
中间件验证
   ↓ 数据库查询
ORM (SQLAlchemy)
   ↓ SQL 执行
MySQL 数据库
   ↓ 返回数据
后端处理和序列化
   ↓ JSON 响应
前端接收并渲染
```

## 🔐 认证机制

```
┌──────────────┐         ┌──────────────┐         ┌──────────────┐
│   前端组件   │────────→│  后端 API    │────────→│   数据库     │
└──────────────┘         └──────────────┘         └──────────────┘
       ↓                        ↓
   发送请求                 验证 Token
   携带 Token               检查权限
       ↓                        ↓
Authorization:            JWT 解码
Bearer <token>            获取用户信息
```

## 🔌 API 接口规范

### 基础 URL
- 开发环境：`http://localhost:8000/api`
- 生产环境：根据部署配置

### 认证方式
```
Authorization: Bearer <JWT_TOKEN>
```

### 响应格式
```json
{
  "success": true,
  "data": { /* 实际数据 */ },
  "message": "操作成功"
}
```

### 主要端点

| 方法   | 路径                      | 功能           |
| ------ | ------------------------- | -------------- |
| POST   | /api/auth/login           | 用户登录       |
| GET    | /api/users/me             | 获取当前用户   |
| GET    | /api/achievements         | 获取成就列表   |
| POST   | /api/achievements         | 添加成就       |
| GET    | /api/courses              | 获取课程列表   |
| POST   | /api/ocr/recognize        | OCR 识别       |

完整 API 文档：http://localhost:8000/docs

## 🚀 启动流程

```
1. 检查环境
   ↓
2. 启动 MySQL 数据库
   ↓
3. 启动后端 FastAPI (端口 8000)
   ├── 初始化数据库连接
   ├── 创建数据表
   └── 启动 Uvicorn 服务器
   ↓
4. 启动前端 Vite (端口 5173)
   ├── 编译 Vue 组件
   ├── 启动开发服务器
   └── 打开浏览器
   ↓
5. 访问应用 (localhost:5173)
```
